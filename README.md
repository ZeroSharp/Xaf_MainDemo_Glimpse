## Example of Glimpse with DevExpress XAF ##

The goal of this project is to configure [Glimpse](http://getglimpse.com) for use with the [DevExpress XAF](http://www.devexpress.com/Products/NET/Application_Framework/) MainDemo sample. The same method can be adapted for use with any XAF website. See the [related blog post](http://blog.zerosharp.com/glimpse-with-devexpress-xaf).

### Dependencies ###

Requires [DevExpress Xaf](http://www.devexpress.com/Products/NET/Application_Framework/). This currently uses version 13.2.5.
